# Banner-QR-negocios

Plantilla web editable para generar banners promocionales con código QR, diseñada para negocios que necesitan material publicitario reutilizable.

## 🚀 Características
- Estructura HTML semántica
- Espacio para logo, texto promocional y código QR
- Fácil de personalizar

## 📂 Archivos principales
- `index.html` - estructura del banner
- `logo de aswa.jpg` - logo de ejemplo
- `README.md` - este archivo
